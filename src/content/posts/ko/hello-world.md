Metsakuur에서 글로벌 엔지니어로 일하고 있다.

Metsakuur는 얼굴인식 보안 솔루션을 다루는 한국 소재 회사다. 국내 업무와 해외 고객사 요구사항에 맞는 시스템을 만들고, 사내 개발에서 글로벌 표준 준수도 담당한다. 특정 포지션에 묶이지 않고 백엔드, 인프라, ML까지 상황에 따라 커버한다.

## Tech Stack

| 분야 | 기술 |
|---|---|
| Machine Learning | PyTorch, TensorFlow, YOLO, Paddle OCR, Tesseract |
| Backend | Spring Boot, Go, FastAPI, Express, Ruby on Rails |
| Frontend | Svelte (선호), React |
| Languages | Python, Go, Java, TypeScript, Rust, Dart |
| Infrastructure | On-premise, Proxmox, Ubuntu, Arch Linux, Docker, LXC |

## 이 블로그에서 다룰 것들

주로 실제로 부딪힌 문제들을 쓴다.

- **Software Engineering** — 설계 결정, 트레이드오프, 삽질 기록
- **ML / AI** — 실무 Vision AI 적용 경험
- **Infrastructure** — 홈서버, 온프레미스, 리눅스
- **Global engineering** — 다국어 환경, 해외 고객사 협업

포스트는 한국어와 영어 둘 다 올릴 예정이다.
